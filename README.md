# 🎮 Something Strange — Gaming Character Animation Website

A futuristic **Gaming Character Animation Website** built using **React.js and modern web technologies**.
This project showcases animated gaming characters, smooth UI interactions, and a modern gaming-themed interface.

🌐 **Live Demo:**
https://something-strange.vercel.app

---

# 🚀 Overview

**Something Strange** is an interactive web project designed to demonstrate modern frontend development skills.
The website focuses on **visual storytelling and animation** while delivering a clean gaming-inspired UI.

The project highlights:

* Interactive character animations
* Smooth UI transitions
* Responsive gaming layout
* Modern web development practices

---

# ✨ Features

### 🎮 Gaming Character Animation

Animated gaming characters displayed with smooth transitions and effects.

### 🖥 Modern UI Design

# ⚙ Installation

Clone the repository

```
git clone https://github.com/lucifer13598/Something-Strange.git
```

Navigate to project folder

```
cd Something-Strange
```

Install dependencies

```
npm install
```

Run development server

```
npm run dev
```

Open in browser

```
http://localhost:5173
```

---

# 🚀 Deployment
